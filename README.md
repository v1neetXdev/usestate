# Counter App

A simple Counter Application built using React.

Features : 

Increase Counter
Decrease Counter
Reset Counter
Fast and responsive UI
Mobile Friendly

Technologies Used : 

- React.js
- JavaScript (ES6)
- CSS3
- HTML5
